GaussSum2

GaussSum2 extracts information from the output of computational chemistry packages and performs a number of useful calculations.

GaussSum2 is platform-independent, and should run on any platform (e.g. Windows, Mac, Linux) for which Python is available.

Its features include:

  • Convolution of the UV-Vis spectr

GaussSum2 uses cclib to parse the output files. cclib can currently handle output files from ADF, GAMESS (US), Gaussian and PC GAMESS.
